Funcionamento CSS: Externo, Interno ou Inline?

  • Externo: criar um arquivo .CSS externo

  • Interno: Utilizar o código CSS na tag HEAD
    Essa não é compartilhável com outras páginas HTML

  • Inline: utilizando a tag "Style"
