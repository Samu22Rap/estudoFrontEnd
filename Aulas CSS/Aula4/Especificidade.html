Especificidades

É o grau de importância de estilos, onde sempre será considerado - se sobresairá - o estilo que estiver em um grau de importância maior. Esses graus são classificados com a notação que remete a do "IP- Internet Protocol" - 0.0.0.0, onde, eles representam:

  1. Inline
  2. Id
  3. Class
  4. Tags

Nessa ordem de importância.

Como exemplo, no código fonte deste site, foi aplicado uma regra de estilo para todos os h4, e como h4 é uma TAG, a ordem de relevância ficaria como 0.0.0.1. Ex:

Este último caso é o caso onde temos o mais completo de todos: o Inline. Onde, além de termos uma declaração de estilo para a tag, classe e id, também temos uma declaração de estilo dentro da linha do código, fazendo com o que está escrito nela se sobresaia sobre qualquer outra declaração realizada.
